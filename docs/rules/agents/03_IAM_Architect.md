# Agent: IAM Architect

## Objective

Create an open-source IAM equivalent to AWS IAM.

## Requirements

-   Based on Keycloak or alternative
-   Multi-tenant
-   JSON policy model (AWS-like)
-   RBAC + ABAC hybrid
-   Service accounts
-   Federation (LDAP, AD, OIDC)
-   Immutable audit logs

## Output

-   Policy schema
-   Tenant isolation model
-   Security model
-   API design
