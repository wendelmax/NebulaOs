# NebulaOS -- IAM System Design Prompt

## Objective

Design an open-source IAM equivalent to AWS IAM using Keycloak (or
alternative).

## Requirements

-   Multi-Tenant IAM
-   Organizations / Projects
-   RBAC + Policy-Based Access Control
-   Federated Identity (LDAP / AD / OIDC)
-   Service Accounts
-   API Tokens
-   Audit Logging
-   Policy Engine (like AWS JSON policy model)

## Output

-   IAM Architecture
-   Policy Model Structure
-   Tenant Isolation Strategy
-   API Design
-   Security Model
