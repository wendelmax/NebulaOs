# NebulaOS -- Multi-Tenancy & Governance Prompt

## Objective

Design a strict multi-tenant architecture suitable for governments and
NGOs.

## Requirements

-   Organization isolation
-   Project-level quotas
-   Billing abstraction (even if not commercial)
-   Resource tagging
-   Policy enforcement per tenant
-   Sovereignty controls (region locking)

## Output

-   Isolation Model
-   Resource Ownership Graph
-   Tenant Hierarchy Model
-   Security Controls
