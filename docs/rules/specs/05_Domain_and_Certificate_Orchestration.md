# NebulaOS -- Domain and Certificate Management Prompt

## Objective

Design a simplified system for:

-   Domain binding
-   Reverse proxy configuration
-   SSL certificate management
-   Let's Encrypt automation
-   Custom certificate upload
-   Automatic renewal
-   DNS integration

## Must Include

-   User-friendly flow
-   Automation pipeline
-   Integration with Kubernetes Ingress
-   Integration with VM-based apps
-   Security considerations

## Output

-   Architecture
-   Workflow
-   APIs
