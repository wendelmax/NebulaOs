# NebulaOS -- Foundational Manifesto Prompt

## Objective

Generate a foundational manifesto for NebulaOS, an open-source sovereign
cloud operating system designed for governments, nonprofits,
institutions, developers, and individuals.

## Core Principles

-   Fully Open Source
-   Vendor Neutral
-   Multi-Cloud & Hybrid Ready
-   Sovereign by Design
-   Multi-Tenant Architecture
-   Infrastructure as Code First-Class Citizen
-   Simple UX for Non-Experts
-   Advanced Control for Engineers

## Deliverables

1.  Vision Statement
2.  Mission
3.  Problem Statement
4.  Core Values
5.  Governance Model (Open Community)
6.  Positioning vs AWS, Azure, GCP
7.  Why NebulaOS Exists

Tone: Institutional, visionary, technically grounded, not marketing
hype.
