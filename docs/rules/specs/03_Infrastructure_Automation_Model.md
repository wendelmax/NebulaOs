# NebulaOS -- Infrastructure Automation Prompt

## Objective

Define how NebulaOS integrates:

-   Terraform (IaC)
-   Crossplane or Juju (Orchestration)
-   Ansible / AWX (Configuration)
-   ArgoCD (GitOps)

-   IaC-first workflow
-   Advanced mode for engineers
-   Wizard-based abstraction for non-technical users
-   Template marketplace (Infrastructure Blueprints)
-   Rollback strategy
-   Drift detection

## Output

-   Workflow Diagram
-   Execution Pipeline
-   State Management Strategy
-   Multi-Cloud Abstraction Method
