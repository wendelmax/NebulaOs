# NebulaOS -- UI/UX System Prompt

## Objective

Design a friendly cloud console similar to AWS Console but simplified
like DigitalOcean.

## User Types

-   Government IT Staff
-   Developers
-   Advanced DevOps Engineers
-   Non-technical Administrators

## Must Support

-   VM provisioning wizard
-   Kubernetes cluster wizard
-   VPC visual builder
-   Domain management
-   Reverse proxy configuration
-   Certificate upload (Proprietary + Let's Encrypt)
-   DNS alignment
-   IAM management UI
-   Cost and Usage dashboard

## Output

-   UX Principles
-   Navigation Structure
-   Module Breakdown
-   Accessibility considerations
